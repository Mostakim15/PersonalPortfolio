Web Design

It refers to the design of websites that are displayed on the internet. It usually refers to the user experience aspects of website development. A good web design is easy to use, aesthetically pleasing, and suits the user group and brand of the website.

Responsive web design

It creates dynamic changes of a website, depending on the screen size and orientation of the device being used to view it. Responsive design relies on proportion-based grids to rearrange content and design elements.

Front-end Development

Front-end web development, also known as client-side development is the practice of producing HTML, CSS and JavaScript for a website or Web Application so that a user can see and interact with them directly. It's easy to learn.
